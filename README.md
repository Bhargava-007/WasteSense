## Overview

WasteSense models a smart waste management platform that manages:
- Waste collection bins and fill-level monitoring
- Garbage collection vehicles and routes
- Recycling centers and waste processing units
- Environmental analytics and reporting

## Repository Structure

```
WasteSense-DSA2/
├── CO1_Trees/
│   └── WasteSenseCO1.java      – AVL Tree (BinID-indexed waste bin records)

```
